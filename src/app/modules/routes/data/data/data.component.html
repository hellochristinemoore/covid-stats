The data set for the graphs and statistics is not being sourced from a data set aggregator website (e.g. worldometer). We try to source data directly from the NICD where possible, but look to a few other sources as well when information from the NICD is latent, difficult to follow or incomplete.

The information is then captured in a database using a Development (free) profile on restdb.io. The data is refreshed when you close and re-open your browser or when you hit the refresh button above. The time of the last sync is displayed below the resfresh button.

Please note there are currently unindicated plotted data nodes where available data sets are incomplete. We plan to make changes in the future so that these are better indicated or alternatively omitted from the data set.
